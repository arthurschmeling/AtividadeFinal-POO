# AtividadeFinal-POO

Este software foi desenvolvido com base nos princípios da Programação Orientada a Objetos (POO), utilizando conceitos fundamentais como encapsulamento, herança e polimorfismo, garantindo uma estrutura de código mais organizada, reutilizável e de fácil manutenção.

O sistema tem como principal objetivo organizar, gerenciar e manipular dados de forma eficiente, oferecendo uma solução simples e intuitiva para o usuário. Sua arquitetura foi projetada para promover separação de responsabilidades, facilitando a escalabilidade e futuras melhorias.

Além disso, o projeto segue boas práticas de desenvolvimento, como:

Estruturação modular do código;
Clareza na definição de classes e métodos;
Facilidade de leitura e entendimento;
Manutenção simplificada.

Com isso, o software se torna uma base sólida tanto para aprendizado quanto para evolução, podendo ser expandido com novas funcionalidades conforme a necessidade.
